NAME

squint - duplicate a monitor output into a X11 window

 

SYNOPSIS

squint [ MONITOR_NAME ]

 

DESCRIPTION

squint is command that duplicates the output of a monitor into a X11 window.

The very purpose of such a gadget is to allow displaying a presentation (or making any kind of demonstration) on multiple monitors in the same time without cloning the entire screen.

Your laptop panel remains fully available to do any kind of work. You can switch back and forth between the two screens just by minimising and restoring squint's window.

 

USAGE

By default squint assumes that the external screen is the monitor having the smallest resolution. This should be the most typical configuration (duplicating a monitor into a smaller one looks less interesting). If you are in that case, then you just need to run the command without any parameter: 

        squint

If the monitor you wish to duplicate is not the smallest one or if there is only one monitor (maybe you enjoy the Larsen effect), then you need to state which monitor (as listed by the xrandr command) shall be duplicated. 

        squint HDMI1

The window displayed by squint provides no interactions at all, it is just a view of the monitor output. In order to interact with the content displayed in this monitor, the pointer must be moved into this monitor.
